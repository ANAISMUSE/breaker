# Phase E 验收清单（可打勾）

范围：`云监测适配层接入` + `安全/合规模块收口`。

## 必要项（上线前必须通过）

- [x] 监测适配层状态接口：`/api/realtime/status` 可返回开关与适配器就绪状态
- [x] 实时监测页展示适配器状态（启用状态/模式/合规确认/输出文件）
- [x] 云监测开关生效：`ENABLE_CLOUD_MONITOR` 可控制真实适配器启停
- [x] 合规闸门生效：未完成合规确认时禁止真实抓取执行
- [x] 审计留痕补齐：监测相关关键动作写入审计日志
- [x] 失败降级策略：真实适配失败可自动回退演示模式并提示

## 可选增强项（建议在 Phase E+）

- [ ] 监测任务化（支持排程、停止、重试）
- [ ] 实时数据流可视化（分钟级指标刷新）
- [ ] 监测告警（阈值、通知渠道、告警抑制）
- [ ] 监测配置模板（平台参数与关键字模板）

## 通过标准

- 必要项全部勾选为通过，且前后端构建成功；
- 监测模块可独立完成“配置 -> 运行 -> 观测 -> 留痕”的最小业务闭环；
- 真实模式与演示模式可在相同页面下可感知切换。
